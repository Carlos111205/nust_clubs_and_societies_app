   - 03_triggers.sql
   - 04_seed_data.sql
 
## Tables
- users: Registered users with profile info
- clubs: All campus clubs
- events: Events created by clubs
- memberships: Who joined which club
- saved_events: Events bookmarked by users
- announcements: Home screen banner messages
